# Cosmetics-Insights-Dashboard
Interactive Tableau dashboard for analyzing cosmetics data with visual insights into brands, categories, prices, ratings, and market trends.
The Cosmetics Insights Dashboard is a Business Intelligence project developed using Tableau Desktop and published on Tableau Public. The project analyzes cosmetics product data and presents meaningful insights through interactive dashboards and stories. It enables users to explore product performance, compare brands, analyze pricing trends, evaluate product ratings, and identify skin-type suitability using dynamic visualizations and filters.

The dashboard is designed to simplify complex datasets into clear and actionable insights, helping businesses understand market trends and assisting customers in making informed purchasing decisions.

Key Features
📊 Interactive Tableau Dashboard
📈 KPI-based Business Insights
🏷️ Brand and Category Analysis
💰 Price Analysis
⭐ Product Rating Analysis
🌿 Skin Type Analysis
🎯 Interactive Filters
📖 Tableau Story for Step-by-Step Insights
☁️ Published on Tableau Public
