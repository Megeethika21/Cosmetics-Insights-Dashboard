# Cosmetics-Insights-Dashboard

## Overview

The **Cosmetics Insights Dashboard** is a data visualization project developed to analyze cosmetic product data and present meaningful business insights through interactive dashboards. The project leverages Tableau to transform raw data into easy-to-understand visualizations that help businesses monitor product performance, identify sales trends, and support strategic decision-making.

In addition to the Tableau dashboards, a responsive project website has been developed to showcase the project objectives, methodology, key insights, and dashboard access in a user-friendly manner.

---

## Problem Statement

The cosmetics industry generates a large volume of product and sales data, making it difficult for businesses to extract meaningful insights using traditional spreadsheets.

Sales managers often spend considerable time analyzing multiple reports to understand product performance and regional sales trends. Similarly, customers find it challenging to compare cosmetic products based on pricing, category, and other attributes due to scattered information.

This project addresses these challenges by providing an interactive dashboard that centralizes data and presents insights through intuitive visualizations.

---

## Objectives

* Develop an interactive Tableau dashboard for cosmetics data analysis.
* Visualize product performance and sales trends.
* Compare products across different categories and brands.
* Enable faster and more informed business decisions through data visualization.
* Present project findings through an interactive Tableau Story.
